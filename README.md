## ⚙️ Integration Details

This project connects mathematical modeling, user interaction, and data visualization within a single Streamlit application.

* **Physics Simulation Engine:** The rocket trajectory is calculated using a simplified physics model based on Newton’s Second Law (F = ma). A step-by-step numerical update is used to convert acceleration into velocity and velocity into altitude while accounting for decreasing rocket mass as fuel burns.

* **Interactive Parameter Control:** Streamlit sliders allow users to modify launch variables such as thrust, fuel mass, and payload mass. When these parameters change, the simulation recalculates instantly and generates a new flight trajectory.

* **Application State Handling:** The application uses Streamlit session state to manage navigation between tabs and maintain consistent interaction flow without resetting the interface on every refresh.

* **Visualization Integration:** Mission datasets are processed using Pandas and visualized with Plotly and Seaborn. These libraries generate interactive charts such as scatter plots, box plots, line charts, and a correlation heatmap to reveal relationships between mission variables.

## 🔬 Research Context – Rocket Physics
The project integrates basic aerospace science concepts to explain rocket motion.

### Newton’s Second Law
The movement of a rocket follows the relationship:

**Force = Mass × Acceleration**

For a rocket to launch successfully, the thrust force generated by the engines must overcome gravitational force and atmospheric drag.

### Mass Reduction During Flight
As fuel burns, the total mass of the rocket decreases. This leads to increasing acceleration during later stages of the launch.

### Atmospheric Drag
Drag is the resistance caused by air particles when the rocket moves through the atmosphere. As altitude increases, the atmosphere becomes thinner, reducing drag and allowing higher speeds.

### 💡 Guiding Questions Answered

* **How does increasing payload affect altitude?**
Ans. Adding more payload increases the rocket’s total mass. Since acceleration depends on the ratio between force and mass, heavier payloads reduce acceleration if thrust remains constant. This can limit the altitude the rocket can reach.

* **Does lower drag improve rocket speed?**
Ans. Yes. As the rocket climbs higher into the atmosphere, air density decreases. This reduces drag and allows the rocket to travel faster with less resistance.
